java基础之集合（一）

集合总览

Collection
|–ArrayList: 底层使用的数据结构是数组，扩容办法是通过new一个新数组（长度增加50%），然后把原数组复制过去。
特点：查询速度快，增删慢。线程不同步，换句话说，效率更高。
|–LinkedList: 底层使用的是链表数据结构。特点：增删速度快，查询速度很慢。
|–Vector:底层使用的是数组。线程同步，jdk1.0以前使用，在jdk1.2以后被ArrayList取代

链表：
链表是一种物理存储单元上非连续、非顺序的存储结构，数据元素的逻辑顺序是通过链表中的指针链接次序实现的。链表由一系列结点（链表中每一个元素称为结点）组成，结点可以在运行时动态生成。每个结点包括两个部分：一个是存储数据元素的数据域，另一个是存储下一个结点地址的指针域.

—————————————————————————————-摘自百度

1,linkedList的特点

  • linkedList的特有方法：
    addFirst();
    addLast();

getFirst();
getLast();
获取元素，但不删除元素。如果集合中没有元素，会出现NoSuchElementException

removeFirst();
removeLast();
获取元素，但是元素被删除。如果集合中没有元素，会出现NoSuchElementException

  • jdk1.6以后出现了替代方法
    offerFirst();
    offerLast();

    peekFirst();
    peekLast();
    获取元素，但是元素不被删除。如果集合中没有元素，会返回null

    pollFirst();
    pollLast();
    获取元素，但是元素被删除。如果集合中没有元素，会返回null

  • 用链表模拟队列
    使用LinkedList模拟一个堆栈或队列的数据结构。

堆栈：先进后出 如同一个杯子
队列：先进先出 如同一个水管

代码示例：

import java.util.*
Class Queue{
  private LinkedList link;

  Queue(){
    link = new LinkedList();
  }

  public void setQueue(Object obj){
    link.addFirst(obj);
  }

  public Object getQueue(){
    return link.removeLast();
  }

  public boolean isNull(){
    retrun link.isEmpty();
  }
}
